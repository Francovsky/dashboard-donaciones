# Dashboard de Donaciones

Dashboard interactivo para visualizar y analizar datos de donaciones recurrentes y esporádicas para una organización sin fines de lucro dedicada al apoyo de inmigrantes en Nueva York.

## Características

- Resumen general de donaciones totales, recurrentes y esporádicas
- Distribución de donaciones por tipo
- Tendencias mensuales de donaciones
- Análisis de número de donaciones por período
- Filtrado por año para análisis específicos

## Tecnologías utilizadas

- React.js
- Recharts para visualizaciones
- Tailwind CSS para estilos

## Instalación y ejecución local

1. Clonar el repositorio:
```
git clone https://github.com/tu-usuario/dashboard-donaciones.git
```

2. Instalar dependencias:
```
cd dashboard-donaciones
npm install
```

3. Ejecutar en modo desarrollo:
```
npm start
```

4. Construir para producción:
```
npm run build
```

## Despliegue

Este proyecto está configurado para desplegarse fácilmente en plataformas como Vercel o Netlify.
